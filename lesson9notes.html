There are two different types of scripting, which are server-side and client-side. Server-side scripting is when the scripts run on the web server and then send the results to the web browser. Client-side scripting, on the other hand, is when the scripts run on the web browser, and it has no interaction with the web server to run the scripts. There are a few languages that are for server-side scripting and those are Perl, PHP, Python, and Ruby. JavaScript can be put in two places, either on its own file or within the HTML file.

The Document Object Model or DOM is the invisible structure of all the documents, the DOM is not a tangible object. JavaScript should be unobtrusive when it's used, which means that it should be usable; if it fails, it should not show an error message and just disappear; the page in which JavaScript is being used should not rely on it to function; and lastly, it should be maintained separately. Images can also be changed depending on how the user interacts with the page. For example, when one is viewing a site that is like a catalog, there are various different pictures, and when the user picks one, that picture is the main one being shown.

JavaScript can manipulate the web browser and documents right in the web browser. It uses the hierarchy of parent and child objects, which means that a parent object is the one that contains other objects, while the child object is the one that is contained within the parent object. The document object has a few properties that can be used to get more information according to the properties, for example, there is one that tells you when the page was last modified or another that involves the site cookies.  Other objects that have properties of their own also include links and anchors, history and written, and more.

Another thing that can also be done by using JavaScript is making layers to help with the positioning of items. Although with HTML and CSS we can position object by object where we want, it is also useful when you can position objects in groups. Besides this, one can also decide when one wants to hide or show objects.  The objects can vary from a border to full-on objects. One can also edit text by modifying it, whether it's by adding or replacing text or nodes.
