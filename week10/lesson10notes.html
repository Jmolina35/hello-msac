JavaScript has a lot of things that make it useful, even as its own language. For example, when you choose a variable name, it is pretty important to follow a few rules and guidelines, like the name can include upper and lowercase letters, numbers and an underscore; the first character must also be a letter; and lastly, there isn't an official limit on how long the name can be, but it is highly recommended that it be at most 1 line long. Another thing that can be achieved is using the keyword "var" to declare a variable. There are other things that can be done with variables that can even be changed to a value. JavaScript can also include different data types. 

Objects are also things that can be used when using JavaScript, these are practically a more complex variables. There are a couple types of variables that can be used depending on what needs to be achieved. There's also a function, which is a group of statements that can be treated as a single unit. And there are arrays, which are number-group units that can be treated as a single unit.

Global variables: The entire script as their scope. Can be used anywhere, even within functions.

Local Variables: Have a single function as their scope. They can only be used in the function they were created in.

To assign values to variables: use the = sign.

Ways to increment and decrement variables: "+=" or "++" are used to increment variables, and "-=" or "--" are used to decrement a variable.

JavaScript data types: Number, boolean, string, and null.

String Object: Used to represent and manipulate a sequence of characters.

Array: Number of group data items that you can treat as a single unit.

To return a value from a function, first you need to set up a function, and then you return the value by using the "return" property.

Object: A more complex kind of variable that can store multiple data items and functions.

Math Object: is a built-in object that includes math constants and functions.

Math.ceil: round up a number to the next integer.
Math.floor: round a number down to the next integer
Math.round: round a number to the nearest integer.

Code example to generate a random number:


            function rand(num) {
            return Math.floor(Math.random() * num) + 1;
            }

Date Object: A built-in object that enables you to work more easily with dates and times.
